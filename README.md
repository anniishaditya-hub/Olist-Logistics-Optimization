# Olist-Logistics-Optimization
End-to-end logistics analytics pipeline using SQL (BigQuery) and Power BI to analyze 100k+ Brazilian e-commerce records.
# Olist Logistics Performance Dashboard
This project analyzes 100k+ rows of Brazilian e-commerce data to identify SLA gaps.

### Tools Used:
* **SQL (BigQuery):** For data joining and time-delta calculations.
* **Power BI:** For executive-level visualization and bottleneck analysis.

### Key Insight:
Average delivery is 11.5 days early, suggesting a massive over-estimation in customer promises.
